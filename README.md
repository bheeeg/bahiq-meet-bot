# Bahiq Meet Bot

## Overview
The Bahiq Meet Bot is designed to enhance your online meetings with various features that simplify and automate common tasks.

## Features
- Automatic meeting scheduling.
- Reminders for upcoming meetings.
- Integration with calendar services.
- Participation tracking.

## Setup
1. **Clone the Repository**: Clone the repository to your local machine using:
   ```bash
   git clone https://github.com/bheeeg/bahiq-meet-bot.git
   ```

2. **Install Dependencies**: Navigate to the project directory and install the necessary dependencies:
   ```bash
   cd bahiq-meet-bot
   npm install
   ```

3. **Configuration**: Update the configuration files with your settings:
   - Open `config.js` and set your API keys and other configuration options.

4. **Run the Bot**: Start the bot using:
   ```bash
   node index.js
   ```

## Usage
- To schedule a meeting, use the command:
   ```bash
   !schedule [meeting details]
   ```

- To get reminders, ensure you have notifications enabled.

- For any issues, please refer to the `issues` section on GitHub or contact the support team.
